Domain model with attributes:

See Creately UML diagram

User Stories:

User (admin) will be able to:

  • Log into the application
  • Create a team
  • Create a project
  • Assign team members to team
  • Assign project to team
  • Assign project tasks to team members
  • View all team members, teams, projects
  • Complete tasks, projects
  • Update/edit projects
  • Delete projects
